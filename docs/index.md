# Proyecto de Integración: XR, IA y Robótica Física
Bienvenido a la documentación técnica del sistema. 

Este proyecto resuelve el problema de traducir instrucciones de voz humanas, capturadas en un entorno de realidad mixta, en trayectorias físicas ejecutadas por un brazo robótico UR3.

## Propósito
El objetivo de este repositorio es documentar el código, la arquitectura de red y el pipeline de modelos de inteligencia artificial utilizados para lograr la comunicación bidireccional entre Unity y los controladores de hardware.
